# 学生作业提交目录

## 目录结构

学生提交的作业将按照**班级**和**日期**自动组织到对应目录:

```
submissions/
├── 七年级1班/
│   ├── 2025-01-09/
│   │   ├── 192.168.1.100_张三_001.json
│   │   ├── 192.168.1.101_李四_002.json
│   │   └── ...
│   ├── 2025-01-10/
│   │   └── ...
│   └── ...
├── 七年级2班/
│   ├── 2025-01-09/
│   │   └── ...
│   └── ...
├── 七年级3班/
├── 七年级4班/
├── 七年级5班/
├── 七年级6班/
├── 七年级7班/
└── 七年级8班/
```

## 文件命名规则

每个提交的作业文件名格式为:
```
[学生IP]_[学生姓名]_[学号].json
```

例如: `192.168.1.100_张三_001.json`

## 文件内容

每个JSON文件包含:
- **studentInfo**: 学生信息(姓名、学号、班级、提交时间、IP地址)
- **topology**: 网络拓扑配置(设备、连接、路由表、DNS、Web服务器等)

## 查看作业

1. 进入对应班级目录,例如 `七年级1班/`
2. 选择日期目录,例如 `2025-01-09/`
3. 打开学生的JSON文件即可查看其作业配置

## 提示

- 目录会在学生第一次提交时自动创建
- 每个班级的作业独立存放,便于分班批改
- 按日期分组,便于查看不同课次的作业
